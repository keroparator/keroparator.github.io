#  Keroparator.me
Bu depo, [keroparator.me](https://keroparator.me) adresinde yayında olan kişisel portfolyomun kaynaklarını içermektedir.

## 🚀 Roadmap
- [ ] CSS değişkenlerindeki tüm pixel birimleri değiştirilecek.
- [ ] Başlıklar, butonlar, logolar tek CSS yapısı altında toplanacak.
- [ ] Mobil uyumluluk geliştirilecek.

## 🛠️ Kullanılan Teknolojiler
- **Dil:** HTML5 ve CSS3
- **Hosting:** GitHub Pages
- **Domain:** Namecheap (Github Education pack aracılığıyla ücretsiz alındı)
- **Tasarım:** Geist ve JetBrains fontları
